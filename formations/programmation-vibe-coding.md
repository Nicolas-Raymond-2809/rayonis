# Programmation et vibe coding

**Accroche :** Apprenez à coder avec l'IA comme copilote
**Durée :** 35-70 heures
**Niveau :** Débutant / Intermédiaire
**Format :** Live & E-learning

## Intention
Ici, on ne mémorise pas la syntaxe. On apprend à diriger l'IA pour qu'elle devienne vos mains, pendant que vous restez l'architecte. Le Vibe Coding est l'art de transformer une intention en code fonctionnel sans friction.

## Modules de la formation

### Module 1 : Les Fondamentaux
Comprendre le Prompt Engineering pour le code, configurer son environnement (VS Code, Cursor, Antigravity), et maîtriser les bases du HTML/CSS assisté.

### Module 2 : Pratique Augmentée
Développement de composants complexes, gestion du JavaScript moderne, et utilisation de GitHub pour le versioning avec l'aide de l'IA.

### Module 3 : Maîtrise & Déploiement
Débogage avancé, optimisation des performances et déploiement continu sur des plateformes comme Vercel.
