# Automatisation n8n

**Accroche :** Crées des workflows complexes
**Durée :** 21-42 heures
**Niveau :** Avancé
**Format :** Live & Workshop

## Intention
Ne soyez plus limité par les intégrations natives. Apprenez à construire votre propre infrastructure d'automatisation avec n8n, l'outil le plus puissant pour connecter vos applications sans limites.

## Modules de la formation

### Module 1 : Logique Node-based
Comprendre les concepts de base de n8n, la manipulation de données JSON et la logique conditionnelle avancée.

### Module 2 : Intégrations API
Se connecter à n'importe quelle API via HTTP Request, gérer l'authentification OAuth2 et les Webhooks.

### Module 3 : Workflows Avancés
Création d'agents autonomes, gestion des erreurs, déploiement sur serveur privé et optimisation des ressources.

## Stack Technologique
- n8n
- Docker
- PostgreSQL
- OpenAI API
- Pinecone
